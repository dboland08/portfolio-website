# Portfolio Website

## Overview

This portfolio website is designed to act as a living resume. A place to show off creativity, techinical skills, and personal projects.

## Tech stack

While this project has NextJS (and thus React), and TailwindCSS built in, the strategy for creating this portfolio is to use a more traditional HTML CSS and JavaScript approach. 
The reasoning behind this decision is that a portfolio site is ultimately simple single page app, so the use of these frameworks doesnt necessarily largly reduce complexity. This approach will help me refine my HTML, CSS and JS skills that can also serve as a reference project that I will be able to look back on in the future.
I also believe that its improtant to master the fundamentals before heavily relying on frameworks, especially since most frameworks are built off of HTML, CSS and JavaScript as prerequisites.
